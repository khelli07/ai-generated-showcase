About this work

Purpose: Promote Indonesian AI-generated arts about food, tourism, and culture on an AI-generated platform/website.

We use 3 technologies here:

  1. Text-to-text: ChatGPT, Github Co-Pilot
  2. Text-to-image: Midjourney (able to text_and_image-to-image as well)
  3. Text-to-video: HourOne.ai
  4. Text-to-audio: TextToMusic.ai

How we made it

We have 4 big components that are generated from AI—that we will cover at its own section.

Website Source Code

We use text-to-text to generate the code of this websites. Questions such as

  1. I want to make an art gallery portfolio, can you give me the folder directory tree for it?
  2. Can you generate the index.html and style.css for me?
  3. Can you fix the css so the photo is on 3x3 grid?

are prompted iteratively to achieve our current web state. However, do note that human intervention are still needed, especially when the code is not working as intended since we might want to undo it. In addition, prompter also needs to understand the AI's response and instructions for the web to work properly.

Pictures and Contents

For the content of images, we use ChatGPT with prompt, more or less, looks like

Please give me contents themed “Indonesian food” for showcase, with the following points.
  • Photo caption
  • Short description
  • Long / detailed description (1 paragraph only)

From this prompt, we use the short description as the input to generate the photo for the gallery. In other words, the flow is prompt to ChatGPT —> get short description —> use short description as an input to Midjourney. The photo caption, short description, and long description can be used as the gallery content.

Video

The video is generated using text-to-video model HourOne.ai by giving text prompt and setting the number of background scenes. The text prompt is retrieved from text-to-text model ChatGPT. An example of the prompt is shown below,

Write a script about Indonesia. Begin with an aerial shot of lush landscapes, transition into diverse traditional dances, and explore iconic landmarks like Borobudur Temple and Mount Bromo. Capture the splendor of pristine beaches and wildlife, including the famous Komodo dragons.

The model then generates relevant images and descriptions text in accordance with the number of background scenes generated. Upon playing the video, an AI presenter is automatically added to elaborate more. Those components are compiled into a presentation resulting in a video of animated scenes with descriptions read by an AI presenter.

Audio

Audio content is generated using text-to-audio model TextToMusic.ai by giving short text prompt describing the music that we want. The short description is also generated using the text-to-text model ChatGPT. One prompt example which result is being used is Indonesia: Treasures of Diversity and Wonder. From one short description, we can get multiple short audios at once. This audio is then played as a background music for the website.
